# Internship Projects

### 1. TaskManager
A Java project to manage daily tasks efficiently.  
**Features:**
- Add, update, and delete tasks
- Mark tasks as completed or pending
- Store tasks in memory (future extension: file or database storage)
- Simple and intuitive console-based interface

**Technologies:** Java, IntelliJ IDEA

### 2. Simple Java Programs
These are beginner-friendly programs to practice Java fundamentals:

| Program | Description |
|---------|-------------|
| Calculator | Performs basic arithmetic operations: addition, subtraction, multiplication, division |
| Fibonacci Series | Generates Fibonacci numbers up to a user-defined limit |
| Palindrome Checker | Checks if a string or number is a palindrome |
| Factorial Calculator | Computes the factorial of a given number |

**Technologies:** Java, IntelliJ IDEA
